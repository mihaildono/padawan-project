# Badges

> "Truly wonderful, the mind of a child is."
> ―Yoda

Track your progress through the Padawan Project by earning badges! Each badge represents mastery of a specific skill or completion of a trial.

## Youngling Phase

### HTML & Foundations

- **Page Builder** - [HTML Part 1](youngling/html-part1.md)
  - Build a complete personal introduction page with proper HTML structure
- **Organized Padawan** - [HTML Part 1 Bonus](youngling/html-part1.md)
  - Master heading hierarchy and code organization with comments
- **Form Master** - [HTML Part 2](youngling/html-part2.md)
  - Create interactive forms with various input types
- **Semantic Sensei** - [HTML Part 2 Bonus](youngling/html-part2.md)
  - Build semantic navigation and forms with validation

### JavaScript Fundamentals

- **Functional Padawan** - [Function Exercises](youngling/exercises/function-tasks.md)
  - Implement classic functional programming: map, filter, and reduce

### TypeScript

- **Type Master** - [TypeScript Basics](youngling/typescript.md)
  - Build a typed task manager with interfaces and proper type safety
