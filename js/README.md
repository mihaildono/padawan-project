Welcome to the Front End Web Development program! This is a structured learning path that will take you from complete beginner to building full-featured React applications.

## Curriculum Overview

### Phase 1: Youngling - HTML & JavaScript Fundamentals

#### 1. [HTML Part 1][html-1]

Learn the basics of HTML and create your first webpage

- HTML structure and syntax
- Common HTML elements (headings, paragraphs, links, images)
- Lists and basic page structure
- **Trial:** Create a personal introduction page

#### 2. [Basic JavaScript Part 1][js-intro]

Introduction to programming with JavaScript

- Primitive types (Numbers, Strings)
- Variables (`let`, `const`)
- Functions
- Conditional statements (`if/else`)
- **Exercises:** [Types][ex-types], [Variables][ex-vars], [Functions][ex-funcs], [Conditionals][ex-cond]

#### 3. [HTML Part 2][html-2]

Advanced HTML concepts and interactivity

- HTML Forms and input types
- Semantic HTML elements
- Tables
- HTML Events
- **Trial:** Create an interactive form page

#### 4. [Basic JavaScript Part 2][js-intro]

Master JavaScript fundamentals

- Arrays and array methods
- Loops (`for`, `while`)
- Objects and properties
- **Exercises:** [Arrays][ex-arrays], [Loops][ex-loops], [Objects][ex-objects]

#### 5. [TypeScript Basics][typescript]

Add type safety to your JavaScript code

- Basic types (string, number, boolean)
- Interfaces and type aliases
- Generics
- Setting up TypeScript
- **Trial:** Build a typed task manager

### Phase 2: Padawan - Styling & React

#### 6. [CSS][css]

Learn to style beautiful and responsive webpages

- CSS basics and selectors
- Flexbox layouts
- Pseudo-classes (`:hover`, `:before`, `:after`)
- Transitions
- **Trial:** Clone Google landing page

#### 7. [React][react]

Build dynamic front-end applications

- React components
- Props and state
- Event handling
- Component lifecycle
- **Projects:** Weather app and more

#### 8. [Node.js & Express][node]

Build a back-end server and REST API

- Setting up a Node.js server
- Express framework basics
- CRUD operations (Create, Read, Update, Delete)
- RESTful API design
- Middleware and error handling
- **Trial:** Build a Task API

#### 9. [Node.js Part 2 - Databases & SQL][node-part2]

Persist data with databases and learn SQL

- Introduction to SQL databases
- Basic SQL commands (SELECT, INSERT, UPDATE, DELETE)
- Setting up SQLite with Node.js
- CRUD operations with a real database
- Table relationships and JOIN queries
- **Trial:** Build a Blog API with Database

### Phase 3: Jedi - Mastery

#### 10. [Final Project][final-project]

Put everything together in a complete full-stack application
